# ATM-10

> Note: This folder contains the source code for the ATM-OS The ATM-OS is a set of programs which run on ComputerCraft: Tweaked specifically the version found within the AllTheMods 10 Modpack.


## Installation
First of all, you will need an advanced computer for this software to run.


## Usage

To install the bootloader, you can use the following command:

```sh
wget run https://raw.githubusercontent.com/AltriusRS/CCT/tree/main/ATM-10/install.lua
```
